# Titanic Dataset Analysis

### Maincrafts Data Science & Analutics Internship - Task 2

## Project Overview

This project analyzes the Titanic dataset using Python. The task includes data cleaning, exploratory data analysis (EDA), and visualization to identify factors that influenced passenger survival during the Titanic disaster.

## Tasks Completed

### Data Exploration and Cleaning

* Loaded the dataset using Pandas.
* Examined dataset structure, columns, and missing values.
* Filled missing values in the Age column using the median.
* Filled missing values in the Embarked column using the mode.
* Removed the Cabin column due to a large number of missing values.

### Data Analysis

* Analyzed survival rates by gender.
* Examined the impact of passenger class on survival chances.
* Calculated survival rates across different age groups.
* Studied the age distribution of passengers.

### Visualizations

* Bar chart of survival rate by gender.
* Bar chart of survival rate by passenger class.
* Bar chart of survival rate by age group.
* Histogram of passenger age distribution.

* task2_titanic_data_analysis.ipynb – Jupyter Notebook containing the complete analysis and visualizations.
